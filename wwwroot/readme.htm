Getting Started

Requirements

dotnet core 3.1 or later

  1. get the project: />git clone https://github.com/raddevus/QRCodeGen
  2. change directory to the local repo and run />dotnet restore
    3. gets all req DLLs
  3. build the project />dotnet build
  4. start the web site />dotnet run
  5. open the link ==> https://localhost:5001/readme.htm
  6. Try out the links below...
